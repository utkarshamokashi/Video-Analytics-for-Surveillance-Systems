Face Detection!!

A tool designed to detect number of faces
Works for real-time monitoring

Tool Used : YOLOv3

How it works

YOLO is an abbreviation for the term ‘You Only Look Once’. It can detect faces and the number of faces in the frame. The main objective is to identify the number of people in one frame. So that if a significantly large number of people are found, an alert is generated. YOLO algorithm employs convolutional neural networks (CNN) to detect objects in real-time. As the name suggests, the algorithm requires only a single forward propagation through a neural network to detect objects.
