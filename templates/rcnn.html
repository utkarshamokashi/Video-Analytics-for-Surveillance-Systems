Tool Used : Faster R-CNN

How it works

Object detection consists of two separate tasks that are classification and localization. R-CNN stands for Region-based Convolutional Neural Network. The key concept behind the R-CNN series is region proposals. Region proposals are used to localize objects within an image. RCNN uses a two-stage approach for object detection. This two-stage approach allows RCNN to have higher accuracy and more precise object detection. Hence we have decided to implement RCNN in our model
